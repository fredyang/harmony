Examples

  1. Documentation App

    This is a single page application. All the content displayed is rendered locally, this rendering starts for <body /> tags. All the templates are external templates. You can edit and delete content, and new content. All change are persisted locally. But it will not affect server side copy. Just click "Undo all change" will reset all content to the server side copy. You can send your change to me by following the following steps.

    1. Open console of chrome developers tools, type copy(hm("api.entries").toJSON())
    2. paste the content to the file doc/doc-json.js in source code.
    3. commit and push change, and send me a pull request.
  2. todos with jsrender and todos with handlebar.js

    This is popular application to evaluate JavaScript MV* framework. There are many implementations hosted at todomvc.com. These two implementations use the same model, but there are some small differences in the view, because the syntaxs of two template engines are different. Feel free to compare these two with other implementations in todomvc.com.
